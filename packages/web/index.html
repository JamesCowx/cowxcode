How it works under the hood

A tight agent loop — from your words to real file changes.

1. You ask a question

Type anything — "explain this project", "write a login function", "find all bugs". The agent receives your message with a system prompt that defines its tools.

user → {"role":"user", "content":"explain this repo"}

2. The model streams a response

Your configured provider (OpenAI, Anthropic, etc.) streams text back. When it needs to act, it emits a tool call as JSON inside a fenced block.

assistant → ```json { "tool": "read_file", "args": {…} } ```

3. CowxCode executes the tool

The engine picks up the JSON tool call, runs it against your real project folder (reads/writes files, runs commands, searches), and feeds the result back to the model.

tool → runs on disk → result injected into messages

4. The loop repeats until done

The model gets the tool result and can call more tools, or deliver a final answer. This continues up to 12 tool rounds per message — then it stops safely.

repeat → until model says "Done!" or max rounds hit

Frequently asked questions

Everything you might wonder about CowxCode.

Is CowxCode really free?
Yes. CowxCode is MIT licensed and 100% free — forever. No subscriptions, no limits on file operations, and no tracking. You only pay for the AI provider you connect (OpenAI, Anthropic, etc.) if they require an API key.
What providers does it support?
OpenAI (and any OpenAI-compatible endpoint), OpenCode Go / Zen, Anthropic Claude, Google Gemini, Mistral, Groq, xAI Grok, Perplexity, OpenRouter, DeepSeek, and local models via Ollama or LM Studio. Anything that speaks the OpenAI chat/completions stream format works.
Is my data safe?
Yes. Your API key is stored locally in your Windows user data folder and is only sent to the provider you configure. Files are accessed only from the project folder you select. Nothing is telemetry'd or uploaded to us.
Can I use it on macOS or Linux?
The core engine runs anywhere Node.js runs. The desktop app is Windows-only for now. You can run the core from source on any OS: clone the repo and use the agent programmatically.
How is it different from opencode?
CowxCode is inspired by opencode but is an independent project with its own codebase, desktop UI, and brand identity. It aims to be a polished, Windows-first alternative with the same provider-agnostic philosophy.
