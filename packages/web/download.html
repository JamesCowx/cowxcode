Get CowxCode

Free, open source, and built for Windows. Two ways to run it.

▣

Installer (recommended)

A standard Windows setup with Start Menu and uninstall support. Best for daily use.

Download .exe
CowxCode-Setup-1.0.0.exe · x64
◈

Portable

A single executable that runs from anywhere. No installation. Great for USB drives.

Download .exe
CowxCode-Portable-1.0.0.exe · x64

Run from source

git clone https://github.com/your-org/cowxcode
cd cowxcode
npm install
npm run build
npm run start:desktop

Requires Node.js 18+ and Windows 10 or later.

CowxCode is provided under the MIT license. Binaries are built with Electron and electron-builder.
