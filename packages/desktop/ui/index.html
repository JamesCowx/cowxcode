Provider Settings

CowxCode is provider-agnostic. Point it at OpenAI, Anthropic, Google, Ollama, LM Studio or any OpenAI-compatible endpoint.
For Ollama use http://localhost:11434/v1. For Google Gemini, leave blank and enter API key.
Stored locally in your user data folder. Never sent anywhere but your provider.
